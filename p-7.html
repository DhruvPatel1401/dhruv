Practical-7 : Method Overriding With Inheritance


class abc
{
    void sum(int a, int b)
    {
        System.out.println("Ans = " + (a + b));
    }
}

class xyz extends abc
{
    void sum(int a, int b)
    {
        System.out.println("Ans = " + (a * b));    
    }
}

class demo
{
    public static void main(String[] args)
    {
        xyz x = new xyz();
        x.sum(8, 4);
    }
}

Output:

Ans = 32
